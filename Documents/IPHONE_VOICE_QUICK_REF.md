# iPhone Voice Commands - Quick Reference Card

## Wake Word
**"Foodie"** - Must say before each command

---

## Quick Commands

### Add Items (NEW!)
- "Add milk"
- "Add chicken breast"
- "Add bread to Walmart"
- "Add eggs to the list"

### Store Switching
- "Show Walmart" / "Show Kroger" / "Show Target"
- "Show all stores"
- "List stores"

### Check Items Off
- "Check milk"
- "Check eggs"
- "Check all"

### Uncheck Items
- "Uncheck milk"
- "Uncheck all"

### Delete Items
- "Delete bread"
- "Clear checked"

### Get Information
- "How many items"
- "What's left"
- "Read list"

### Search
- "Search for chicken"
- "Clear search"

### Help
- "Help"

---

## Example Session
```
"Foodie, add milk"
   → "Added milk"
   → Milk appears in Dairy category

"Foodie, show Walmart"
   → Switches to Walmart store

"Foodie, how many items"
   → "12 items, 8 remaining"

"Foodie, check milk"
   → Milk gets checked

"Foodie, what's left"
   → "7 items remaining"
```

---

## Setup (First Time)
1. Settings → Voice Control
2. Tap "Enable" (grants permissions)
3. Toggle "Voice Commands" ON
4. Choose listening mode
5. Start using!

---

## Tips
✅ Speak clearly at normal volume
✅ Say item names as they appear in list
✅ Use continuous mode while shopping
❌ Don't forget to say "Foodie" first
❌ Don't use when backgrounded

---

## Battery Saver
Use **Single-Command Mode** when not actively shopping
(Settings → Voice Control → Continuous Listening OFF)
